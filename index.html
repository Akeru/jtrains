JTrains

Free, offline japanese railways route finder for Android.

Features

Free

No ads, no crippled/lite version.

Offline

No data network required.

Comprehensive

Covers more than 630 lines and around 9000 stations.

Felica support

Support scanning and displaying major fare cards.

Localized

Readings provided for all lines and stations.

Compatible

Runs on Android 2.3.3 and up.

Contact

For any bug report, feature request or database correction please see the Github issue tracker.
